# Frontend Design Plugin

Generates distinctive, production-grade frontend interfaces that avoid generic AI aesthetics. Adapted for the ezLander project across both the SwiftUI macOS app and the Next.js marketing website.

## What It Does

Claude automatically uses this skill for frontend work. Creates production-ready code with:

- Bold aesthetic choices aligned with ezLander's warm coral-amber design system
- Distinctive typography and color palettes
- High-impact animations and visual details
- Context-aware implementation for both SwiftUI and Next.js

## Usage

```
"Redesign the email compose view"
"Build a new onboarding flow"
"Create a pricing page for the website"
"Add a dark mode calendar widget"
```

Claude will choose a clear aesthetic direction and implement production code with meticulous attention to detail, respecting the existing design tokens and patterns.
